# 概要

文字をイタリック体・斜体で表示します。少なくとも日本語サイトをコーディングする場合は多用しませんので、必要になれば調べる程度で大丈夫でしょう。

なお、斜体は通常の書体を機械的に傾けたもので、イタリック体は傾いた文字になるようにデザインされたものを言います。利用しているフォントに斜体・イタリック体が存在しない場合、ブラウザが自動で傾いた表示に補正します。
