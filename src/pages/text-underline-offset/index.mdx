# 概要

下線のオフセット値（離れる距離）を指定します。

## 注意事項

`text-decoration-xxx`系のプロパティと異なり、[`text-decoration`](/text-decoration)でまとめて設定することはできません。

また、性質上[Example](#Example)のように`text-decoration: underline;`と一緒に用います。

# Example

```html
<p>The <span class="line">Tokyo Olympics</span> were held in 2021.</p>
```

```css
.line {
  text-decoration-line: underline;
  text-underline-offset: 4px;
}
```
