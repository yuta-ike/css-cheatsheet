# 概要

文字色を指定することができます。指定する値は[色の指定方法](/article/color)を参照してください。

{

<p className="text-sm text-twhite-50 opacity-70">
  初見だと「`color`ってなんの色?」となりますが、文字色のことです。
</p>
}

# Example

```html
<p class="text">
  あの<span class="emphasis">イーハトーヴォ</span>のすきとおった風、
  夏でも底に冷たさをもつ青いそら、うつくしい森で飾られた <span class="emphasis">モリーオ市</span>、
  郊外のぎらぎらひかる草の波。
</p>
```

```css
.text {
  color: #dbe1e9;
}
.emphasis {
  color: #38bdf8;
}
```
