# 概要

アウトラインと、アウトラインで囲う要素の間の距離を指定します。

`outline`のショートハンドには含まれませんので、個別に指定する必要があります。

# Example

```css
.box {
  outline: 1px solid hotpink;
  outline-offset: 2px;
}
```
