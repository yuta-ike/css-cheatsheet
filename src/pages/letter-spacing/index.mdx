# 概要

文字の間隔を指定します。負の値も指定できますが、デザインが崩れる可能性があります。
見出し、太字、強調箇所などで文字間隔を広げる場合に用いられます。

このサイトでは、`hello`のようなコード表記の部分で、`letter-spacing: -0.025em;`を指定しています。

## よく利用される値

`px`, `em`を抑えておけば良いでしょう。`em`を利用すると「0.1 文字分のスペース（`0.1em`）」のように、文字サイズからの割合で指定することができます。
