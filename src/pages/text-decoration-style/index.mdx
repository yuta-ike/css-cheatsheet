# 概要

傍線の種類を指定します。`solid`（実線）, `double`（二重線）, `dotted`点線, `dashed`破線, `wavy`（波線）の 5 種類を指定できます。

通常は[`text-decoration`](/text-decoration)プロパティで、傍線に関する値をまとめて設定することが多いです。
